📌 Abstract

K Programming Language is a modern, lightweight and beginner-friendly scripting language designed with simplicity and clarity in mind. It introduces a space-separated syntax model where every token is clearly readable, eliminating unnecessary complexity. The language allows execution even when some lines fail, improving debugging efficiency and learning speed.

📖 Introduction

K Programming Language follows a unique principle where each word is separated by space, making programs readable and structured. It aims to balance simplicity and flexibility while avoiding the complexity of traditional languages. K is designed not to replace existing languages but to provide an easier entry point for beginners and experimental developers.

🎯 Design Philosophy

  • Simplicity – Easy for beginners
  • Clarity – Clean readable syntax
  • Flexibility – Freedom in coding style
  • Growth – Expandable in future versions

⚙️ Technical Overview

  • Execution: Line-by-line
  • Syntax: Space-separated
  • File Extension: .k
  • Platforms: Windows, Linux, macOS, Android
  • Dependencies: None

💻 Basic Syntax Example

display > "Hello World"
a = 10
b = 20
result = a + b
display > result

🔁 Control Flow

a = 1
loop a != 3
. a = a + 1
. display > "Hello"
/loop

🧩 Functions

fun greet
- display > "Hello from K!"
/fun

call greet

📂 File Operations

write file.txt data
read file.txt
copy file.txt backup.txt
